# Python Programming Practice – Functions & Object-Oriented Programming

Repository ini berisi kumpulan **hands-on practice Python** yang mencakup konsep dasar pemrograman hingga penerapan **Object-Oriented Programming (OOP)** menggunakan Jupyter Notebook.

Project ini bertujuan untuk memahami bagaimana konsep dasar Python digunakan untuk membangun program yang lebih terstruktur dan modular.

---

# Project Overview

Pada project ini dilakukan beberapa latihan pemrograman Python yang meliputi:

- Tipe data dasar pada Python
- Pembuatan dan penggunaan function
- Konsep dasar Object-Oriented Programming (OOP)
- Implementasi class dan object
- Penerapan prinsip utama OOP

Seluruh latihan dilakukan menggunakan **Python dan Jupyter Notebook**.

---

# Files in This Repository

Berikut beberapa notebook yang terdapat dalam project ini:

### 1. Week2_Tipe_Data.ipynb
Notebook ini berisi latihan dasar mengenai **tipe data dalam Python**, seperti:

- Integer
- Float
- String
- Boolean
- List
- Tuple
- Dictionary

Latihan ini bertujuan untuk memahami bagaimana Python menyimpan dan mengelola berbagai jenis data.

---

### 2. Hands_On_Function_in_Python.ipynb

Notebook ini berisi latihan pembuatan **function dalam Python**.

Beberapa hal yang dipraktikkan antara lain:

- Membuat function sederhana
- Menggunakan parameter
- Mengembalikan nilai dengan `return`
- Menggunakan function untuk memecah program menjadi bagian yang lebih modular

---

### 3. Hands_On_OOP_Paradigm_Concepts.ipynb

Notebook ini berfokus pada **konsep dasar Object-Oriented Programming (OOP)**.

Beberapa konsep yang dipelajari:

- Class
- Object
- Attribute
- Method
- Pembuatan object dari class

Tujuannya adalah memahami bagaimana OOP digunakan untuk mengorganisasi kode program.

---

### 4. Hands_On_Main_Principles_of_OOP.ipynb

Notebook ini membahas **prinsip utama dalam Object-Oriented Programming**, yaitu:

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

Melalui latihan ini dipelajari bagaimana prinsip-prinsip tersebut diterapkan dalam Python menggunakan class dan object.
