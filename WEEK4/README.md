# Supervised Learning (Regression)

## 📌 Deskripsi Project

Project ini merupakan implementasi **Supervised Learning (Regression)** menggunakan dataset *USA Housing*.
Tujuan utama dari project ini adalah untuk memprediksi harga rumah berdasarkan beberapa fitur seperti income, age, dan jumlah kamar.

Project ini dibuat sebagai bagian dari pembelajaran Machine Learning 

---

## 📂 Struktur File

* `Supervised_Learning_Hands_On_Regresi.ipynb` → Notebook utama berisi proses analisis dan modeling
* `USA_Housing.csv` → Dataset yang digunakan
* `Regression.pdf` → Materi pembelajaran terkait regression

---

## ⚙️ Tools & Library

Project ini menggunakan:

* Python
* pandas
* numpy
* matplotlib
* seaborn
* scikit-learn

---

## 🔍 Tahapan Pengerjaan

1. Import dataset menggunakan pandas
2. Exploratory Data Analysis (EDA)
3. Data preprocessing
4. Split data (training & testing)
5. Training model regression
6. Evaluasi model

---

## 📊 Dataset

Dataset yang digunakan adalah **USA Housing Dataset** yang berisi informasi:

* Average Area Income
* Average Area House Age
* Average Area Number of Rooms
* Average Area Number of Bedrooms
* Area Population
* Price (target)
